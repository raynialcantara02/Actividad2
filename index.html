1. ¿Cuáles son las principales funciones de un kernel en la arquitectura de un sistema operativo y cómo interactúa con el hardware?

El kernel es el núcleo del sistema operativo y tiene varias funciones clave, como la gestión de memoria, la gestión de procesos, la gestión de dispositivos de hardware y la seguridad del sistema. Interactúa con el hardware actuando como intermediario entre el software y los componentes físicos del sistema. Controla cómo se accede y se utiliza el hardware, asegurando que los programas se ejecuten de manera eficiente y sin interferencias.

3. ¿Cómo se gestionan los procesos y cuál es la importancia de la multitarea en la eficiencia del sistema?

Implementación de cortafuegos (firewalls) para controlar el tráfico de red entrante y saliente, filtrando paquetes según reglas predefinidas para evitar accesos no autorizados. Uso de sistemas de detección y prevención de intrusiones (IDS/IPS) para monitorear la actividad de la red y detectar y prevenir actividades maliciosas. Actualización regular del sistema operativo y de las aplicaciones para corregir vulnerabilidades conocidas y mantener la seguridad del sistema. Implementación de políticas de acceso y autenticación, como contraseñas seguras, autenticación de dos factores y control de acceso basado en roles, para limitar el acceso solo a usuarios autorizados. En entornos empresariales, el uso de herramientas de gestión de parches y configuraciones para mantener un estado de seguridad coherente en todos los dispositivos de la red.
